# 秘法富翁棋网页版

这是可以部署成公网链接的网页版本。把整个文件夹上传到任意支持静态网站的 HTTPS 服务后，就能把网址发给微信好友。

## 新版玩法亮点

- 困难模式：AI 开局资源更多，并会主动施放法术。
- 新增法力、护盾、诅咒、强化骰等状态。
- 掷骰前可施放秘银盾、地契共鸣、厄运咒。
- 地产加入派系连锁，集齐同派系后租金更高。
- 地产最高可升到 4 级，后期租金压力更强。
- 奇遇卡更丰富，包含免费升级、法力泉涌、护盾和压制效果。
- 画面强化为原创奇幻酒馆卡牌风：镶金卡框、法阵、法力水晶和发光边框。

## 文件

- `index.html`：网页入口。
- `game.js`：游戏逻辑和 Canvas 画面。
- `netlify.toml`：Netlify 静态部署配置。
- `vercel.json`：Vercel 静态部署配置。

## 最简单的发布方式

### Netlify Drop

1. 打开 `https://app.netlify.com/drop`。
2. 把 `arcane-tycoon-web` 文件夹拖进去。
3. 等它生成 HTTPS 链接。
4. 把链接发给微信好友。

### Vercel

1. 新建一个 Vercel 项目。
2. 上传或关联这个文件夹。
3. Framework Preset 选 `Other`。
4. 发布后复制 HTTPS 链接。

### 自己的网站空间

把 `index.html` 和 `game.js` 上传到同一个目录，然后访问该目录的 HTTPS 地址即可。

## 注意

- `file:///...` 本地路径只能你自己电脑打开，发给别人不行。
- 微信里直接打开网页链接时，建议使用 HTTPS。
- 这个版本是网页游戏，不等同于微信小游戏正式版；微信小游戏正式版仍然需要走微信平台上传、审核、发布。

# 秘法富翁棋

这是一个大富翁式微信小游戏原型，画面是原创的奇幻酒馆卡牌桌游风：木质桌面、羊皮纸棋盘、镶金卡框、法力水晶、中心法阵和卡牌式按钮。

## 怎么运行

1. 打开微信开发者工具。
2. 选择“小游戏”项目。
3. 导入本文件夹：`arcane-tycoon-wechat`。
4. AppID 可先使用测试号或游客模式。
5. 编译后点击底部按钮掷骰开始。

也可以直接打开 `web-preview.html` 做浏览器预览，方便先看玩法和布局。

## 当前玩法

- 困难模式：玩家 560 金币开局，AI 650 金币开局，并会主动施法。
- 掷骰沿棋盘前进，路过起点获得 120 金币。
- 玩家拥有金币、法力、护盾、诅咒等状态。
- 掷骰前可施放法术：秘银盾、地契共鸣、厄运咒。
- 空地产可以购买；自己的地产可以升级到 4 级。
- 地产拥有派系，同派系集齐后触发租金连锁。
- 踩到对手地产需要支付租金，护盾可抵消租金和税收。
- 棋盘包含税收、奖励、随机奇遇和传送门格子。
- 奇遇包含强化骰、免费升级、护盾、诅咒、法力泉涌等效果。
- 任意一方金币跌破 0 时游戏结束。

## 文件说明

- `game.js`：小游戏主逻辑和 Canvas 绘制。
- `game.json`：微信小游戏配置。
- `project.config.json`：微信开发者工具项目配置。
- `web-preview.html`：浏览器预览壳。
